# 🥤 Classic Shakes - Full Stack MERN Project

A premium milkshake ordering app built with the MERN stack.

## 🌐 Live Demo
- **Frontend:** https://classic-shakes.vercel.app
- **Backend API:** https://classic-shakes-backend.onrender.com
- **GitHub:** https://github.com/Sravani021447/Classic-Shakes

## 👨‍💼 Admin Features
- Register & Login
- Add/Delete Shakes from Menu
- View all customer orders
- Update order status (Pending/Preparing/Delivered)

## 👤 Customer Features
- Register & Login
- Browse shake menu
- Add shakes to cart
- Place orders
- View order history & status

## 🛠️ Tech Stack
- **Frontend:** React, Vite, React Router, Axios
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Authentication:** JWT
- **Deployment:** Vercel + Render
